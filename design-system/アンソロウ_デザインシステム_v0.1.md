# アンソロウ デザインシステム v0.1

> **トーン**: Anthropic Claude-style（白基調・コーラルアクセント・セリフ見出し）
> **対象**: 業務SaaS / 管理ツール — 不動産現場の知性的業務運用
> **位置づけ**: インフラレベル業務SaaSの設計基盤

---

## 1. 設計思想（Why this design）

「**人が哲学的な問いに集中できる**」ために、UIは情報を**減らす**設計。
- 余白で語る。装飾で語らない。
- 数字は等幅で読める。文章はセリフで品位を持たせる。
- アクセントは1色（珊瑚 #CC785C）のみ。判断を分散させない。

## 2. カラーパレット

| 役割 | トークン | 値 | 使い所 |
|---|---|---|---|
| 紙基調 | `--色-紙-基調` | `#FAF9F7` | メイン背景 |
| 紙純白 | `--色-紙-純白` | `#FFFFFF` | カード／モーダル |
| 紙くすみ | `--色-紙-くすみ` | `#F2EFEA` | セクション区切り |
| 紙淡墨 | `--色-紙-淡墨` | `#E8E4DC` | ボーダー |
| 墨本体 | `--色-墨-本体` | `#1F1E1D` | 主要テキスト |
| 墨中 | `--色-墨-中` | `#44423D` | 副次テキスト |
| 墨淡 | `--色-墨-淡` | `#6B6863` | キャプション |
| **珊瑚本体** | `--色-珊瑚-本体` | `#CC785C` | **唯一のアクセント** |
| 珊瑚濃 | `--色-珊瑚-濃` | `#B86347` | ホバー／プレス |
| 珊瑚淡 | `--色-珊瑚-淡` | `#F4E4DC` | バッジ／選択 |

機能色（成功・警告・危険・情報）は淡背景＋濃文字の組合せ。

## 3. タイポグラフィ

- **見出し**: Tiempos Headline → Copernicus → Noto Serif JP（フォールバック）
- **本文**: Styrene B → Inter → Noto Sans JP（フォールバック）
- **数字**: JetBrains Mono（金額・KPI 数値は等幅）

スケール: 12 / 14 / 16 / 18 / 22 / 28 / 36 / 48 px（Major Third）。

## 4. 余白・グリッド

8pxグリッド固定。`--余白-1` (4px) 〜 `--余白-9` (96px)。
- カード内部 padding は `--余白-5` (24px) を基準。
- セクション間は `--余白-7` (48px)。

## 5. コンポーネント原則

| コンポーネント | 高さ | 角丸 | 影 |
|---|---|---|---|
| ボタン主要 | 40px | 12px | 影なし |
| カード | 任意 | 16px | 影-1 |
| モーダル | 任意 | 16px | 影-モーダル |
| 入力フィールド | 40px | 12px | フォーカス時 珊瑚3pxリング |
| タグ／バッジ | 24px | 9999px（丸） | なし |

## 6. アクセシビリティ

- コントラスト比: 主要テキスト ≥ 14:1（AAA）／珊瑚アクセント vs 白 ≥ 4.5:1（AA）
- フォーカスリング: 珊瑚極淡3px、輪郭2pxで二重視認性
- キーボード操作: 全インタラクションが Tab / Enter / Esc で完結
- 文字最小: 12px（補助情報のみ。本文は 14px 以上）

## 7. 命名規則

CSS変数・クラスは**和文**を優先（不動産業界の専門用語と整合）。
コードシンボル（JSコンポーネント名）は英語 PascalCase。

## 8. バージョン管理（厳守原則）

```
[v0.1] 初版（本書）
  ↓ 動作確認 → 樋口承認 → 保存 or 改善
[v0.2] 改訂版
[v0.3] 安定版（最終3版まで保存・復元可能）
```

変更時フロー: **変更提案 → 動作確認 → 2択（保存/改善）→ 適用**。
保存先: `versions/v0.x/` にスナップショット保管。
